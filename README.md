# Real-Time Chat Application

A full-stack real-time messaging application that enables users to communicate instantly through WebSockets. The system supports real-time message delivery, persistent chat history, and responsive user interfaces.

## Features

* Real-time messaging between multiple users
* WebSocket communication using Socket.io
* Persistent chat storage using MongoDB
* REST APIs for user sessions and message handling
* Responsive frontend built with React
* Efficient backend built with Node.js and Express
